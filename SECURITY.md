# Security Policy

LedgerLite is privacy-first: all data stays on device, with no accounts or
servers. If you discover a security or privacy issue, please report it
privately rather than opening a public issue.

- Open a [private security advisory](https://github.com/tqakdev/ledger_lite/security/advisories/new), or
- Email the maintainer.

Please include steps to reproduce and the affected version. We aim to respond
within a few days.
